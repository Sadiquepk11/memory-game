# Memory Game using Zybo Z7-10

A memory-based game implemented on the Zybo Z7-10 development board using the Zynq-7000 SoC architecture. The project demonstrates hardware-software co-design through the integration of the Processing System (PS) and Programmable Logic (PL) using AXI GPIO interfaces.

The hardware platform was designed in Vivado, while the game logic was developed in Embedded C using Vitis. The game generates a sequence of LED patterns that the player must memorize and reproduce using push buttons. With each successful round, the sequence length increases, making the game progressively more challenging.

This project showcases concepts such as Zynq PS-PL interfacing, AXI GPIO communication, embedded C programming, real-time user interaction, and FPGA-based embedded system design.
